# 网页纵向切片

《6/16》是一款以雨天校园时间循环为舞台的 AVG 纵向切片。学生们被困在 6 月 16 日，通过调查、对质与跨循环保留的事实逐步接近真相。

## 直接试玩

生产构建已生成在 `dist/`。在本目录运行：

```bash
python3 -m http.server 4173 -d dist
```

然后打开 <http://127.0.0.1:4173>。

## 已实现

- 标题画面、完整序章与第一章《缺席者》纵向切片
- 可跳过的开场动画、开始界面选单、章节选择与声音/自动速度设置
- 第二轮证明、资料室调查、悠真对质与旧连廊替代事故
- 两张可跨循环保留的事实卡及事实列表
- 三处统一美术方向的雨中校园环境
- 程序化雨声、铃声、广播杂音与记忆残响提示
- 对话记录、快进、自动播放、声音开关和浏览器本地存档
- 键盘 `Enter`、空格和右方向键推进
- 35 KB 首屏预览与压缩场景图，标题稳定后在后台预载后续地点

完整回归脚本位于 `scripts/qa-capture.mjs`，会生成 1280×720 试玩截图、交互结果和并排视觉对照。

## 开发

依赖可用时运行：

```bash
npm install
npm run dev -- --port 4173
```

当前视觉 QA 状态见 [design-qa.md](design-qa.md)。

线上试玩由 GitHub Pages 从 `main` 分支的 `docs/` 目录发布。更新版本前先运行路线审计和生产构建，再同步 `dist/` 到 `docs/`。
